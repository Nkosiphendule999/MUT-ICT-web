# User Guide - MUT ICT Website

## For Students

### How to Register
1. Go to Register page
2. Fill in your details
3. Submit the form

### How to Login
- Students: Use MUT Student form with student number
- External: Use External User form with email
- Admin: Use MUT Student form with ADMIN001 / Ncobela@05

### How to Book a Consultation
1. Go to Consultation page
2. Select a lecturer
3. Choose date and time
4. Enter module and reason
5. Submit

### How to Use the Chatbot
1. Click chatbot icon or go to Chatbot page
2. Type your question
3. Get AI-powered responses

### How to View/Edit Profile
1. Click "My Profile" in navigation
2. View your information
3. Edit personal details
4. Change password if needed

## For Admins

### How to Manage Events
1. Login as Admin
2. Go to Manage Events
3. Add/Edit/Delete events

### How to View Dashboard
1. Go to Admin Dashboard
2. View statistics and analytics
